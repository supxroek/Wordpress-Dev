 # WordPress

 ## Getting Started

 - run `dotenvx run -- docker-compose up -d`
 - open `http://localhost:8080`
 - login with `username` / `password`
 - success

 ## Usage

 ### wp-content
 - files are stored in `wp-content`
 - wordpress files are stored in `wp-content`
 - wordpress is accessible at `http://localhost:8080`

 ### wp-database
 - database is stored in `wp-database`
 - database credentials are stored in `mysql`
 - database is accessible with `phpMyAdmin` at `http://localhost:8081`
